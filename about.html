A little about me

I'm a Computer Science graduate from Sinhgad Academy of Engineering, Pune, specialising in Java Full Stack Development. I enjoy building backend systems that are clean, scalable, and actually useful — and I'm equally comfortable working on the frontend when the project calls for it.

I'm actively looking for entry-level Java Developer or Full Stack roles where I can contribute, learn fast, and grow alongside a strong team.
